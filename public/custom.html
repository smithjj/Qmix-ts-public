Refractive-index model used for custom crystals
n(λ,T) = sqrt(abs(A + B / (λ² - C) - Dλ²)) + Δn(λ,T)
λ is in micrometers. Enter one row of A B C D coefficients for each principal index. Extra values in a row are ignored by this page.
Nonlinear model used by the uniaxial Qmix calculation
d1 = A cosθ + B sinθ
d2 = C cos²θ + D sin(2θ)
These values are stored with the custom crystal and used by the standard uniaxial phase-matching calculation. Biaxial custom nonlinear tensors are not fully represented by this simplified form.
